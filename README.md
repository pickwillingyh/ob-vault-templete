# Obsidian 日记模板

这是两套实用的日记模板，分别适用于个人成长记录和工作日志。

## 模板列表

| 模板 | 文件 | 用途 |
|------|------|------|
| 📝 个人日记 | `Templates/Daily.md` | 记录自我发展、待办事项、未来规划 |
| 💼 工作日志 | `Templates/Work.md` | 记录工作内容、会议、项目进度 |

---

## 📝 Daily.md - 个人日记模板

### 模板特点

- **自我发展记录**：记录每天做过的有利于个人成长的事
- **待办事项管理**：分层次管理今日、本周、本月待办
- **未来探索**：记录想做但不明确的事项，帮助理清思路
- **每日反思**：总结亮点、改进空间和明日计划

---

## 💼 Work.md - 工作日志模板

### 模板特点

- **工作记录**：按任务条目记录今日完成的工作内容和耗时
- **会议记录**：记录会议时间、参与人、议题和行动项
- **项目追踪**：表格化跟踪进行中的工作状态和进度
- **问题管理**：记录遇到的问题、严重程度和解决方案
- **规划提醒**：重要事项、截止日期和跟进事项提醒

## 使用方法

### 1. 安装 Templater 插件

1. 在 Obsidian 中安装 `Templater` 社区插件
2. 插件设置中：
   - 指定模板文件夹：`Templates`
   - 启用 "Daily notes" 功能

### 2. 配置日记文件夹

建议创建以下文件夹结构：

```
📁 Daily/
  📁 2024/
    📁 01-January/
      📄 2024-01-15.md
```

### 3. 快捷键设置

在 Templater 插件设置中为不同模板设置快捷键：
- `Ctrl/Cmd + D` → `Templates/Daily.md` (个人日记)
- `Ctrl/Cmd + W` → `Templates/Work.md` (工作日志)

或使用命令面板 `Ctrl/Cmd + P` → "Templater: Insert Template" 选择模板。

## 模板语法说明

模板使用了 Templater 插件的语法：

- `{{date}}`：当前日期时间
- `{{date:YYYY-MM-DD}}`：格式化日期
- `{{date:dddd}}`：星期几
- `{{date:WW}}`：周数

如果未安装 Templater，可以手动编辑替换这些变量。

## 推荐插件搭配

| 插件 | 用途 |
|------|------|
| Templater | 模板渲染 |

| Dataview | 数据查询和统计 |
| Tasks | 任务管理和筛选 |
| Tags | 标签管理 |
